キャラクター管理、D66シナリオ進行、ダイスロール、ログ保存といった、プレイに必要な機能がブラウザひとつに集約されています。このツールは一切の情報収集を行いません。

🚀 はじめに

ダウンロード後適切なフォルダに解凍してください

HTMLファイルを開く: お使いのWebブラウザ（Chrome, Edge, Safari推奨）で RLHcompanion‗ver1.0.html を開いてください。

シナリオの準備: シナリオを読み込ませなくても使用することができますが、テンプレートに従ったCSⅤファイル を読み込ませることで、ダイス出目や特殊ボタンに合わせて自動的にシナリオテキストが表示されるようになります。同梱の 「D66シナリオ見本.csv」がテンプレートになりますので、ご活用ください。どんなものか試してみたい方は、拙作「夏越の宵」でファイルを用意していますのでお試しください。

🛠 主な機能

1. キャラクター管理

キャラクターの追加・削除：キャラクターを自由に追加・削除できます

ステータス操作: 技量点、生命点、金貨などをリアルタイムに更新。ステータスは削除することもできます。

ログ: ステータス操作をログとして記録します。

個別保存: 表示中のキャラクターデータのみをJSON形式で書き出し、他の端末やセッションへ持ち越すことができます。

2. 判定とダイスロール

RLH判定: 「技量判定」は現在の技量点を、「能力判定」はドロップダウンで選択中の副能力値を参照し、自動でダイスを振って合計を出します。個数を入力すると、同じ修正値、同じ目標値で同時に複数のダイスを振ることができます。従者分をまとめて振るときにご利用ください。

宝物ロール: シナリオCSVに宝物テーブルが設定されていれば、その宝物ロールを、なければ「黄昏の騎士」準拠の宝物表ロールをおこないます。「黄昏の騎士」準拠の場合、自動で結果の中のロールも連続して行います（1d6枚の金貨の場合、1d6まで行う）。ロール結果が7以上の場合、「魔法の宝物表」への誘導メッセージが表示されます。修正値の加減算も可能です

汎用ダイス: 1D3/1D6/D66/D33を網羅。修正値の加減算も可能です。

魔法の宝物ロール：シナリオCSVに魔法の宝物テーブルが設定されていればそのロールを、なければ1d6の魔法の宝物表ロールをおこないます。

3. D66 GRID とシナリオ進行

GRID連動: グリッドをクリックすると色が変わるだけでなく、マップタイルへの記録、シナリオCSVが読み込まれていればその出目に対応するシナリオテキストをログへ出力します。

特殊イベント: 「中間」「最終」ボタンを押すと、シナリオテキストが出力されるとともに、マップタイルの空きマスにその履歴が自動的に記録されます。

回数枠：ラジオボタンで1回目の冒険～3回目の冒険の記録をすることができます。

4. シナリオCSVの活用

同梱の 「D66シナリオ見本.csv」 を参考に自作シナリオを作成できます。
type 列に pro (プロローグ), start (冒険のはじまり), mid (中間イベント), clear (冒険の達成), finish (最終イベント) などを指定することで、ボタンひとつで演出を呼び出せます。
周を指定することで、同じ出目でも違うテキストを送信することができます。
もちろんD66以外のシナリオにも対応。シナリオタイルは一本道モードで3周13枚目まで対応。グリッドはd66表示のみ、タイルも強制的に13枠表示されます。シナリオにないグリッドを押したときには出目のみ表示されます。分岐管理はできません。
CSV形式でシナリオを配布することができます。その場合、ローグライクハーフの製作に関する利用規約を満たすことがむずかしいため、必ず通常の方法での配布に付帯する形で配布（頒布）してください。現在利用規約で満たすことが難しい項目は、ライセンス表記、参照の記載です。
